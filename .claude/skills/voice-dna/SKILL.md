# Voice DNA — Voice Discovery from Photography Content

> Measure the photographer's voice. Don't assume it.

## Purpose

Analyzes existing product pages and gallery text to extract the actual voice patterns. Feeds like-a-human and voice-audit with measured data instead of guessed rules.

## When to Fire

- On `/voice-dna` command
- When voice feels like it's drifted
- When adding a new content type

## Analysis

Sample the best product descriptions and gallery text. Extract:

- **Visual precision frequency:** how often light, time, weather are named specifically
- **Restraint level:** average description length, ratio of said to unsaid
- **Technical mention style:** when technique appears, how it's worn
- **Emotional vocabulary:** which feelings are named directly vs. implied
- **Sentence rhythm:** compression-release patterns specific to photography writing
