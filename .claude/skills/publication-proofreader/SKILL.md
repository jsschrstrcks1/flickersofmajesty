# Publication Proofreader — Photography E-Commerce

> The last pass before buyers see it.

## When to Fire

- On `/proofread` command
- Before publishing product pages
- After batch updates

## Checks

### Typography
- [ ] Curly quotes (" ") not straight (" ")
- [ ] Curly apostrophes (' not ')
- [ ] Em-dashes (—) not double hyphens
- [ ] No double spaces

### Product Page Polish
- [ ] Prices formatted consistently ($XX.XX)
- [ ] Size options listed in consistent order (small → large)
- [ ] Print medium names consistent (Canvas, Metal, Framed — capitalized)
- [ ] Image alt text describes the actual photograph, not the product SKU
- [ ] No placeholder text or TODO items

### Photography-Specific
- [ ] Photographer's name spelled consistently
- [ ] Location names verified
- [ ] Print dimensions match what's actually offered
- [ ] "Flickers of Majesty" not "flickers of majesty" in brand references
